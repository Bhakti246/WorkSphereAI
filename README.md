# WorkSphere AI

An AI-powered Human Resource Management System (HRMS) built with modern web technologies.

## Features

- Employee Management
- Attendance Management
- Payroll Management
- Leave Management
- Reports & Analytics
- Employee Portal
- Notifications
- Company Settings
- Dashboard with Charts
- JWT Authentication

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Chart.js
- React ChartJS 2
- Lucide React

### Backend
- Django
- Django REST Framework
- PostgreSQL
- JWT Authentication


### Frontend

```bash
npm install
npm run dev
```
